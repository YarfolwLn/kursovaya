# 🎁 Wishlist App

Яркое приложение для создания списка желаний с системой рейтинга подарков.

## 🌈 Особенности

- **Яркий дизайн** в голубых и розовых тонах
- **Регистрация и вход** пользователей
- **Личный кабинет** для управления желаниями
- **Система рейтинга** от 1 до 5 звезд
- **Хранение данных** в localStorage
- **Адаптивный дизайн** для всех устройств

## 🚀 Запуск

### Способ 1: С Node.js сервером (рекомендуется)

1. Установите зависимости:
```bash
npm install
```

2. Запустите сервер:
```bash
npm start
```

3. Откройте в браузере: `http://localhost:5003`

### Способ 2: Прямое открытие файлов

Просто откройте `index.html` в браузере.

## 📱 Использование

1. **Регистрация**: Создайте аккаунт на странице регистрации
2. **Вход**: Войдите в свой аккаунт
3. **Добавление желаний**: В личном кабинете нажмите "Добавить желание"
4. **Рейтинг**: Укажите важность подарка от 1 до 5 звезд
5. **Управление**: Просматривайте, редактируйте и удаляйте желания

## 🎨 Дизайн

- **Основные цвета**: Ярко голубой (#00D9FF) и розовый (#FF69B4)
